# Go-Todo-API 🚀

基于 Go + Gin 构建的 RESTful 待办事项管理 API 服务。

## ✨ 特性

- ✅ **完整的用户认证系统** - JWT Token 认证，支持访问令牌和刷新令牌

- ✅ **完整的 Todo CRUD** - 创建、读取、更新、删除待办事项

- ✅ **智能查询筛选** - 支持状态、优先级、关键词等多维度筛选

- ✅ **标签管理** - 为待办事项添加分类标签

- ✅ **统一响应格式** - 标准化的 API 响应结构

- ✅ **配置化管理** - 支持多环境配置（开发、生产）

- ✅ **结构化日志** - 使用 Zap 进行高性能日志记录

- ✅ **API 文档** - 集成 Swagger 自动生成文档
